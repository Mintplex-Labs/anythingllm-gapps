# AnythingLLM Google Apps Script

This is a Google Apps Script that allows you to use AnythingLLM in various Google Apps via Google Apps Script.

This does **not** require you to make a Google Cloud API key or anything complex like that. At worst, you might have to copy-paste a few lines of code into your Google Apps Script project. The indended use case is for small businesses or individuals who want to use AnythingLLM with Google Apps without having to deal with the complexity of setting up a Google Cloud API key - which is fraught with errors and frustrations for the everyday user.
